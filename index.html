Aplikasi siap digunakan untuk ujian atau absen harian.
Absen Harian Mulai absensi koordinat sekolah
Cara Menggunakan
  1. Pastikan koneksi internet stabil sebelum beraktivitas.
  2. Untuk Absen Harian, langsung klik menu Absen Harian.
  3. Untuk Absen PKL, Anda harus memasukkan NIS valid yang terdaftar sebagai peserta magang aktif.
  4. Untuk Ujian, mintalah QR Code soal ujian dari pengawas kelas.

Verifikasi Siswa PKL

Silakan masukkan Nomor Induk Siswa (NIS) Anda untuk membuka gerbang absensi PKL.
