Aplikasi siap digunakan untuk ujian atau absen harian.
Absen Harian Mulai absensi koordinat sekolah Absen PKL Mulai absensi di lokasi tempat magang/industri
Cara Menggunakan
  1. Pastikan koneksi internet stabil sebelum beraktivitas.
  2. Untuk Absen, klik menu Absen Harian atau Absen PKL untuk langsung menuju sistem peta radius lokasi Anda.
  3. Untuk Ujian, mintalah QR Code soal ujian dari pengawas kelas.
  4. Kerjakan ujian dengan jujur. Sistem mendeteksi kecurangan.
